Open PDF Reports in browser

Main features

  • Preview PDF reports in browser instead of downloading
  • Download reports later from browser if you need (filename will be preserved)
  • Note: you must enable pop-up windows for Odoo when prompted!
  • Hint #1: Manage all Odoo messages easily and much more effective using free Mail Messages Easy module
  • Hint #2: Manage reordering rules easily using free Advanced Auto Reordering Rules Control module
  • Hint #3: Manage multiple email addresses and phone number for Contact using Improved Contacts module
