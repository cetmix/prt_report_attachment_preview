Image Grid View and Gallery View

  • Adds Image Grid and Gallery Views to Odoo
  • Both views are defined and customized via xml definitions like regular Tree, Kanban or Form Odoo views
  • Use Image Grid View to display multiple items (Products, Partners etc)
  • Click on item's image to open selected item in Gallery view mode
  • Click on item's caption to open item in Form view mode
  • Use Gallery View to display single item. You can switch to/from Form view it you want to edit this item using Odoo view selection buttons
  • Use buttons on both sides of the Gallery View for next/previous image
  • Click the image in Gallery View to open it full screen
  • Click the image in full-screen mode to close it
  • Swipe for next/previous images when using device with touch screen
  • Tested on Odoo Community v11.0
  • Russian language translation
  • English, German and Russian language support
